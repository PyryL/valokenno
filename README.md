# Valokenno

## Setup development environment

<!--
Versions I used:
* IDE 2.3.6
* esp32 v3.2.1
* VL53L0X v1.3.1
-->

First, install Arduino IDE.
In Boards manager, install `esp32` by Espressif Systems.
In Library manager, install `VL53L0X` by Pololu.
Select `AI Thinker ESP32-CAM` from the board list.
